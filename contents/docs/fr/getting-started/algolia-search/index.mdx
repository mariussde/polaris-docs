title: Commencer avec Algolia DocSearch

---

[Algolia DocSearch](https://docsearch.algolia.com/) fournit une expérience de recherche rapide et pertinente pour les sites de documentation. Il alimente la recherche dans ce projet (**PolarisDocs**). Suivez ce guide pour l'intégrer dans votre documentation.

## Demander DocSearch

Algolia fournit **DocSearch** gratuitement pour les projets open-source. Pour faire une demande :

1. Visitez la [Demande DocSearch](https://docsearch.algolia.com/apply/).
2. Remplissez le formulaire avec les détails de votre documentation :
   - **L'URL de votre site web**
   - **Les sélecteurs pour les titres et le contenu**
   - **Le lien de votre dépôt GitHub** (si applicable)
3. Soumettez votre demande. Une fois approuvée, Algolia fournira :
   - **ID d'Application**
   - **Clé API de Recherche**
   - **Nom d'Index**

## Installer DocSearch dans Votre Projet

Pour intégrer DocSearch, installez le package requis :

```sh
npm install @docsearch/react@3
```

## Créer un Composant de Recherche
Dans votre projet React, créez un Composant de Recherche en utilisant `@docsearch/react` :

```jsx
// src/components/Search.js
import { DocSearch } from '@docsearch/react';
import '@docsearch/css';

export function Search() {
  return (
    <DocSearch
      appId="VOTRE_APP_ID"
      indexName="VOTRE_INDEX_NAME"
      apiKey="VOTRE_CLE_API_RECHERCHE"
    />
  );
}
```
