Questa cartella contiene il codice principale dell'applicazione per Next.js, gestendo layout, routing e pagine di contenuto specifiche. È organizzata per supportare sia le sezioni `docs` che `blog`, con pagine e layout dedicati per ogni sezione.

### lib

Questa cartella contiene funzioni helper e utilità utilizzate in tutta l'applicazione, come logica di parsing Markdown e routing. Queste utilità aiutano a mantenere la base di codice pulita e organizzata separando la funzionalità comune.

### Aggiuntivo

- **`package.json`**: Contiene metadati sul progetto, dipendenze e script per costruire ed eseguire l'applicazione.
- **`tailwind.config.ts`**: Configura Tailwind CSS, permettendo personalizzazione di colori del tema, font e breakpoint responsive specifici per questo progetto.

Questa struttura organizza il tuo progetto in un modo che supporta scalabilità e manutenibilità, facilitando la navigazione e il lavoro su diverse sezioni dell'applicazione.
