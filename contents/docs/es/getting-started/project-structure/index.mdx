## Visión General de la Estructura del Proyecto

### app

Esta carpeta contiene el código principal de la aplicación para Next.js, gestionando layouts, enrutamiento y páginas de contenido específicas. Está organizada para soportar tanto las secciones `docs` como `blog`, con páginas y layouts dedicados para cada sección.

### lib

Esta carpeta contiene funciones auxiliares y utilidades utilizadas en toda la aplicación, como lógica de parsing de Markdown y enrutamiento. Estas utilidades ayudan a mantener la base de código limpia y organizada separando la funcionalidad común.

### Adicional

- **`package.json`**: Contiene metadatos sobre el proyecto, dependencias y scripts para construir y ejecutar la aplicación.
- **`tailwind.config.ts`**: Configura Tailwind CSS, permitiendo personalización de colores de tema, fuentes y breakpoints responsivos específicos para este proyecto.

Esta estructura organiza tu proyecto de una manera que soporta escalabilidad y mantenibilidad, facilitando la navegación y el trabajo en diferentes secciones de la aplicación.
