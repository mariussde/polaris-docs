## Descripción General del Proyecto

**PolarisDocs** es más que solo una plantilla de documentación. Es un **kit de herramientas completo** diseñado para el desarrollo web moderno. Las características principales incluyen:

- **Soporte para Markdown & MDX:** Escribe documentación fácilmente en Markdown, con la opción de incluir componentes interactivos a través de MDX.
- **Temas Personalizables:** Diseñado con un tema minimalista inspirado en ShadCN que es fácil de estilizar.
- **Optimizado para SEO:** Cada página está lista para SEO, asegurando que los motores de búsqueda puedan encontrar y clasificar tu contenido.
- **Bloques de Código Interactivos:** Bloques de código bellamente estilizados y específicos del lenguaje para una experiencia de lectura mejorada.

### Características Principales

| **Característica**                   | **Descripción**                                          |
|-------------------------------|----------------------------------------------------------|
| Soporte MDX                   | Escribe documentación interactiva con MDX.                |
| Páginas Anidadas                  | Organiza contenido en una estructura jerárquica anidada.    |
| Sección de Blog                  | Incluye una sección de blog dedicada.                        |
| Paginación                    | Divide contenido en múltiples páginas.                     |
| Resaltado de Sintaxis           | Resalta código para mejor legibilidad.                   |
| Resaltado de Líneas de Código y Títulos | Resalta líneas específicas con títulos descriptivos.      |
| Bloques de Código Interactivos       | Visualización de código específica del lenguaje e interactiva.          |
| Componentes de Markdown Personalizados             | Incrusta componentes personalizados y reutilizables en tu documentación.          |
| Generación de Sitio Estático        | Genera un sitio estático de alto rendimiento.                |
| Optimizado para SEO                 | Estructurado para una indexación óptima en motores de búsqueda.           |


## Tecnología y Bibliotecas

PolarisDocs aprovecha tecnologías de vanguardia para asegurar rendimiento y flexibilidad:

- **Next.js 14** - El potente framework de React optimizado para producción.
- **Tailwind CSS** - Estilos utility-first para diseños rápidos y limpios.
- **Shadcn-UI** - Componentes elegantes y accesibles para una apariencia pulida.
- **next-mdx-remote** - Habilita soporte MDX para contenido Markdown dinámico e interactivo.
